### Server Planning
#### How to run:
##### Running with local php interpreter:
You have to have PHP7.1 or higher version.

Installation:
```bash
composer install
```
Running tests
```bash
./vendor/bin/phpunit
```

##### Running with docker-compose:
```bash
docker-compose up
```
